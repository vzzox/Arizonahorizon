NewToyers — Clans

Правила создания кланов
1. Название клана должно быть уникальным.
2. Запрещены оскорбительные и нецензурные названия.
3. Создатель клана несёт полную ответственность.
4. Клан создаётся только через официальную команду.
Правила кланов
1. Участники обязаны соблюдать общие правила проекта.
2. Запрещены внутренние конфликты и провокации.
3. Лидер отвечает за действия участников.
4. Нарушения приводят к санкциям против клана.
Правила создания каналов клана
1. Каналы создаются только для клановой деятельности.
2. Название канала должно соответствовать тематике.
3. Запрещён спам и реклама.
4. Модерация канала — обязанность клана.
Правила поведения в суде
1. Уважительное общение обязательно.
2. Запрещены крики, флуд и перебивания.
3. Решение суда не оспаривается публично.
4. За неуважение — мгновенное наказание.
