Правила создания кланов
1. Название клана должно быть уникальным.
2. Запрещены оскорбительные и нецензурные названия.
3. Создатель несёт полную ответственность.
4. Клан создаётся только официальным способом.
Правила кланов
1. Соблюдение правил проекта обязательно.
2. Запрещены конфликты и провокации.
3. Лидер отвечает за участников.
4. Нарушения → санкции.
Правила создания каналов клана
1. Только для клановой деятельности.
2. Название — по теме.
3. Спам запрещён.
4. Ответственность на клане.
Правила поведения в суде
1. Уважение обязательно.
2. Без флуда и перебиваний.
3. Решение суда окончательное.
4. Нарушения караются сразу.
